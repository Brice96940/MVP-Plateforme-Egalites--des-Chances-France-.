# Égalité des Chances — MVP (France)

Plateforme minimale (MVP) visant à réduire les inégalités d'accès à l'information, aux ressources et aux opportunités.

## 🎯 Objectif

Donner un point d’appui structuré aux étudiants, jeunes diplômés et profils sans réseau :
- Orientation
- Méthode
- Accès aux ressources
- Opportunités
- Outils pratiques

## 🧩 Structure

- Frontend : GitHub Pages (site vitrine)
- Backend : Base Notion centralisée
- Inscription : Google Form
- Déploiement : Gratuit

## 🚀 Vision

Réduire le hasard dans les parcours d’études et d’emploi.
Structurer l’accès à l’information.
Créer un levier d’autonomie.

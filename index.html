FAQ

Gratuit vs Premium : c’est quoi la différence ?

Le gratuit = base utile pour démarrer. Le premium = templates + trackers + système complet + gains de temps (Notion PRO).

Non. Ce sont des ressources pédagogiques et organisationnelles. Tes résultats dépendent de ton contexte et de ton travail.

Le payant est-il accessible sur GitHub ?

Non. GitHub reste public et ne contient pas les contenus premium. L’accès premium passe par Gumroad + validation (formulaire).
